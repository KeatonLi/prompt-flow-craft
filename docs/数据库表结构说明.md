# 数据库表结构说明

## 表总览

| 表名 | 说明 |
|------|------|
| agent_prompts | Agent 提示词 |
| skill_prompts | Skill 提示词 |
| prompt_category | 提示词分类（8个预设） |
| prompt_tags | 标签 |
| shared_prompt | 分享的提示词 |

---

## 1. agent_prompts（Agent 提示词表）

存储 AI Agent 提示词

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键，自增 |
| name | VARCHAR(100) | Agent 名称，如"写作助手"、"代码审查员" |
| role_description | TEXT | 角色定位描述，描述 Agent 的核心角色 |
| capabilities | TEXT | 核心能力，列出 Agent 能做什么 |
| behaviors | TEXT | 行为规范，定义 Agent 的行为准则 |
| communication_style | VARCHAR(50) | 对话风格：professional/friendly/concise/detailed/casual |
| generated_prompt | TEXT | 生成的完整提示词内容 |
| author_id | BIGINT | 作者ID（预留，暂不使用） |
| author_nickname | VARCHAR(100) | 作者昵称（游客发布时使用） |
| like_count | INT | 点赞数 |
| view_count | INT | 浏览数 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

**索引：**
- `idx_author_id` - 作者ID索引
- `idx_like_count` - 点赞数索引（用于热门排序）
- `idx_created_at` - 创建时间索引（用于最新排序）

---

## 2. skill_prompts（Skill 提示词表）

存储 Tool/Function Calling 格式的 Skill 定义

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键，自增 |
| name | VARCHAR(100) | Skill 名称，使用 snake_case，如 get_weather |
| description | TEXT | 功能描述，详细说明 Skill 的功能和使用场景 |
| skill_type | ENUM('api','function','webhook','data') | Skill 类型 |
| method | VARCHAR(10) | HTTP 方法（API类型时使用）：GET/POST/PUT/DELETE |
| endpoint | TEXT | API 端点 URL |
| parameters | JSON | 输入参数定义，JSON 格式 |
| output_description | TEXT | 输出描述，说明返回值结构 |
| generated_prompt | TEXT | 生成的完整提示词内容 |
| author_id | BIGINT | 作者ID（预留，暂不使用） |
| author_nickname | VARCHAR(100) | 作者昵称 |
| like_count | INT | 点赞数 |
| view_count | INT | 浏览数 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

**索引：**
- `idx_author_id` - 作者ID索引
- `idx_like_count` - 点赞数索引
- `idx_created_at` - 创建时间索引

---

## 3. prompt_category（分类表）

预设的 8 个提示词分类

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键，自增 |
| name | VARCHAR(100) | 分类名称，如"办公效率"、"内容创作" |
| description | VARCHAR(500) | 分类描述 |
| icon | VARCHAR(50) | 分类图标 emoji |
| color | VARCHAR(20) | 主题颜色 |
| sort_order | INT | 排序顺序 |
| is_system | BIT | 是否系统预设（1=系统预设不可删除） |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

---

## 4. prompt_tags（标签表）

用户可创建的标签

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键，自增 |
| name | VARCHAR(50) | 标签名称（唯一） |
| color | VARCHAR(20) | 标签颜色 |
| usage_count | BIGINT | 使用次数统计 |
| is_system | BIT | 是否系统预设 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

---

## 5. shared_prompt（分享表）

存储用户分享的提示词，无需登录即可发布

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键，自增 |
| description | TEXT | 提示词描述 |
| prompt_content | TEXT | 提示词内容 |
| author_nickname | VARCHAR(100) | 作者昵称 |
| author_contact | VARCHAR(500) | 联系方式（邮箱、微信等） |
| like_count | INT | 点赞数 |
| view_count | INT | 浏览数 |
| source_prompt_id | BIGINT | 来源历史记录ID（关联 agent_prompts 或 skill_prompts） |
| delete_token | VARCHAR(32) | 删除令牌（UUID，作者凭此删除） |
| last_like_time | DATETIME | 最后点赞时间（用于防刷冷却） |
| created_at | DATETIME | 发布时间 |
| updated_at | DATETIME | 更新时间 |

**索引：**
- `idx_delete_token` - 删除令牌索引
- `idx_like_count` - 点赞数索引
- `idx_created_at` - 创建时间索引

---

## SQL 建表语句

```sql
-- ============================================
-- Prompt Flow Craft 数据库初始化脚本
-- ============================================

CREATE DATABASE IF NOT EXISTS prompt DEFAULT CHARSET utf8mb4 COLLATE utf8mb4_unicode_ci;
USE prompt;

-- Agent 提示词表
CREATE TABLE agent_prompts (
  id BIGINT NOT NULL AUTO_INCREMENT PRIMARY KEY,
  name VARCHAR(100) COMMENT 'Agent名称',
  role_description TEXT COMMENT '角色定位描述',
  capabilities TEXT COMMENT '核心能力',
  behaviors TEXT COMMENT '行为规范',
  communication_style VARCHAR(50) COMMENT '对话风格',
  generated_prompt TEXT COMMENT '生成的完整提示词',
  author_id BIGINT COMMENT '作者ID',
  author_nickname VARCHAR(100) COMMENT '作者昵称',
  like_count INT DEFAULT 0 COMMENT '点赞数',
  view_count INT DEFAULT 0 COMMENT '浏览数',
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
  INDEX idx_author_id (author_id),
  INDEX idx_like_count (like_count),
  INDEX idx_created_at (created_at)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='Agent提示词';

-- Skill 提示词表
CREATE TABLE skill_prompts (
  id BIGINT NOT NULL AUTO_INCREMENT PRIMARY KEY,
  name VARCHAR(100) COMMENT 'Skill名称',
  description TEXT COMMENT '功能描述',
  skill_type ENUM('api','function','webhook','data') DEFAULT 'api' COMMENT 'Skill类型',
  method VARCHAR(10) COMMENT 'HTTP方法',
  endpoint TEXT COMMENT 'API端点',
  parameters JSON COMMENT '输入参数定义',
  output_description TEXT COMMENT '输出描述',
  generated_prompt TEXT COMMENT '生成的完整提示词',
  author_id BIGINT COMMENT '作者ID',
  author_nickname VARCHAR(100) COMMENT '作者昵称',
  like_count INT DEFAULT 0 COMMENT '点赞数',
  view_count INT DEFAULT 0 COMMENT '浏览数',
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
  INDEX idx_author_id (author_id),
  INDEX idx_like_count (like_count),
  INDEX idx_created_at (created_at)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='Skill提示词';

-- 分类表
CREATE TABLE prompt_category (
  id BIGINT NOT NULL AUTO_INCREMENT PRIMARY KEY,
  name VARCHAR(100) NOT NULL COMMENT '分类名称',
  description VARCHAR(500) COMMENT '分类描述',
  icon VARCHAR(50) COMMENT '分类图标',
  color VARCHAR(20) COMMENT '主题颜色',
  sort_order INT COMMENT '排序顺序',
  is_system BIT DEFAULT b'0' COMMENT '是否系统预设',
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间'
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='提示词分类';

-- 标签表
CREATE TABLE prompt_tags (
  id BIGINT NOT NULL AUTO_INCREMENT PRIMARY KEY,
  name VARCHAR(50) NOT NULL UNIQUE COMMENT '标签名称',
  color VARCHAR(20) COMMENT '标签颜色',
  usage_count BIGINT DEFAULT 0 COMMENT '使用次数',
  is_system BIT DEFAULT b'0' COMMENT '是否系统预设',
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间'
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='提示词标签';

-- 分享表
CREATE TABLE shared_prompt (
  id BIGINT NOT NULL AUTO_INCREMENT PRIMARY KEY,
  description TEXT COMMENT '提示词描述',
  prompt_content TEXT COMMENT '提示词内容',
  author_nickname VARCHAR(100) COMMENT '作者昵称',
  author_contact VARCHAR(500) COMMENT '联系方式',
  like_count INT DEFAULT 0 COMMENT '点赞数',
  view_count INT DEFAULT 0 COMMENT '浏览数',
  source_prompt_id BIGINT COMMENT '来源历史记录ID',
  delete_token VARCHAR(32) COMMENT '删除令牌',
  last_like_time DATETIME COMMENT '最后点赞时间',
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
  INDEX idx_delete_token (delete_token),
  INDEX idx_like_count (like_count),
  INDEX idx_created_at (created_at)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='分享的提示词';
```
